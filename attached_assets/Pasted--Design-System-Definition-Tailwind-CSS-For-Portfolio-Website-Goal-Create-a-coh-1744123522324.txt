## 🎨 Design System Definition — Tailwind CSS (For Portfolio Website)

**Goal:**  
Create a **cohesive and scalable design system** using **Tailwind CSS** for my personal portfolio website. This system should ensure **visual consistency** across all pages and components while remaining clean, modern, and responsive.

---

## 🧱 Key Elements to Define

### 1️⃣ **Color Palette**
- Define a **primary**, **secondary**, and **accent** color.
- Include **neutral tones** (for backgrounds, text, borders).
- Specify **hover** and **active** states for buttons and links.
- Ensure **accessibility and contrast** (meet WCAG standards).

💡 _Example: Tailwind `extend.theme.colors` config_

---

### 2️⃣ **Typography**
- Choose 1 or 2 complementary **Google Fonts** (e.g., sans-serif for body, serif or display for headers).
- Define:
  - Font sizes (e.g., `text-sm`, `text-lg`, `text-2xl`)
  - Font weights (e.g., `font-light`, `font-semibold`)
  - Line heights and letter spacing
- Ensure typography is **responsive** and scales well across screen sizes.

💡 _Use Tailwind utilities like `leading-relaxed`, `tracking-wide`, etc._

---

### 3️⃣ **Spacing Scale**
- Create a consistent spacing system using Tailwind’s `spacing` utilities (`p-`, `m-`, `gap-`, `space-`, etc.).
- Define a base unit (e.g., 4px = `1`)
- Apply spacing rules for:
  - Section padding and margins
  - Grid gaps and component spacing
  - Responsive breakpoints (`sm`, `md`, `lg`, `xl`)

💡 _Extend or customize spacing in `tailwind.config.js` if needed._

---

### 4️⃣ **Border Radius & Shadows**
- Define consistent **rounded corners** (`rounded-md`, `rounded-xl`) for buttons, cards, inputs.
- Establish **shadow levels** (light, medium, hover emphasis) for elevation effects.

---

### 5️⃣ **Component Style Guidelines**
Apply the design system rules to:

- ✅ Navigation bar
- ✅ Section headers
- ✅ Buttons (primary, secondary, disabled)
- ✅ Cards (e.g., project showcase)
- ✅ Footer and contact form
- ✅ Forms and inputs

Use consistent:
- Color usage
- Font styles
- Padding/margins
- Border radius and shadows

---

## ⚙️ Deliverables Expected
- Suggested **`tailwind.config.js` customizations** (for colors, fonts, spacing)
- Example class names and reusable utility combinations
- Optional: Create utility-first components or Tailwind plugin recommendations

---

## 📐 Overall Design Vibe
- Clean and modern
- Developer-friendly
- Lightweight and responsive
- Subtle interactivity (for future animation layering)
